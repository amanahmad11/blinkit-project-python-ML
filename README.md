# Blinkit Sales Analysis

## Objective
To analyze sales data and identify key factors affecting performance.

## Key Steps
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Machine Learning (Random Forest)

## Results
- Identified top-performing categories
- Built a predictive model with R2 score ~0.54

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
